Material Plan with Change Management


When Change Management is enabled, the Material Plan report becomes an extended version of the standard Material Plan interface. It includes additional columns and controls that allow Material Planning Managers to monitor, review, and act on Change Requests submitted by Functional Material Planners and Venue Logistics Managers.


New columns, which appear in the Material Plan:


Column

Description

Change request

The new Requested quantity submitted by the Functional Material Planner.

Delta

The difference between the originally Requested quantity and the Change Request quantity.


Positive means that Requested quantity will be reduced.

Negative means that Requested quantity will be increased.

Status

Indicates the current state of each Change Request.


Material Planning Managers can perform the following actions directly in the Material Plan report:


  • Review Change Requests on a Materials Request (line ID) basis;
  • Approve, Decline, or Partially Approve Change Requests individually;
  • Monitor Deltas to evaluate redistribution opportunities.


Once a Change Request is reviewed, the Material Planning Manager’s decision — whether Approved, Declined, or Partially Approved — triggers a specific scenario in the request lifecycle, automatically updating the Material Request line, quantities, and statuses according to the defined Approval Logic.

Actions & Interface


1. Go to Material Planning - Overview in the Navigation Panel.



2. You will see all non-archive Events created in the system. Functional Material Planners and Venue Logistics Managers will only see the Events they have been granted access to.



3. Click on the Material Plan icon in the Action Column next to selected Event or click on the Event for which you want to access Material Plan.



4. Once inside the selected Event, click on the Material Plan icon in the Action Bar.


Alternatively, you can get to this interface by selecting Event - Overview in the Navigation Panel under the Material Planning section. 



5. You will see consolidated Material Plan report for selected Event with additional Change Management columns.


Analytics (in orange) show total count and sum of all Materials on all pages and will adjust when using Search and Filters.



6. Use the Status column filter to display only active Change Requests — those with the status Pending or Partially Confirmed.


The Change request column shows the new quantity submitted by the user, while the Delta column indicates the difference between the original and requested quantities — a negative value represents a decrease, and a positive value represents an increase.


To process a Change Request, click the Confirm or Decline icon in the Action Column. A status message will appear to show error or confirm that the selected action has been performed successfully.
