Import Materials Request


SM2ARTA allows users to import Material Requests from other Events created in the system (not the same Event). This feature is particularly useful for reusing historical data or baseline planning — for example, importing Material Requests from a previous event and then adjusting them for current requirements.


Material Requests can only be imported into empty Logical Spaces. If a Logical Space already contains Materials, the Import option will be inactive.


If there are differences between the Catalogues (Materials) configured for the source and destination Events, only the Materials that exist in the destination Event’s catalogue will be imported. Materials that are not available in the destination Event catalogue will not be imported, and a status message will be displayed listing all Materials that were excluded from the import.


This catalogue check is performed across all catalogues configured for the destination Event, not only the specific Project catalogue.


When importing Materials for Project A, if certain items are not present in Project A’s catalogue but are available in another catalogue (for example, Project B’s catalogue) that is also configured for the destination Event, those materials will still be imported.


Actions & Interface


1. Go to Material Planning - Overview in the Navigation Panel.



2. You will see all non-archive Events created in the system. Functional Material Planners and Venue Logistics Managers will only see the Events they have been granted access to.



3. Click on the Event in which you want to Import Materials Request.



4. You will see the list of Venues already added to this Event. 


Alternatively, you can get to this interface by selecting Event - Overview in the Navigation Panel under the Material Planning section. 



5. If the Venue in which you want to Import Materials Request is already on the list, proceed to step 8. Otherwise, click Add venue to add a new one.



6. Proceed with this step only if the required Venue is missing from the list.


Because the Material Planning Manager can create a Materials Request under any Project, the Project name must be selected. Functional Material Managers assigned to this Project will be able to view and edit this request. 


Venue Logistics Managers can only view Material Requests. They do not have permission to add, edit, or delete Material Requests.




7.Once the Project is selected, you will see a list of Venues that have not yet been added to the selected Event under this Project. Venues that have already been added will not appear in this interface.


Use the checkboxes in the Selection Bar to select the Venues you want to add. You can use the Search and Filters to narrow down your selection. Once finished, click Add venue.


Note that selections are not saved between pages — if you navigate to another page, your previous selections will be cleared.


To avoid losing selections, finalize your actions on the current page before moving to the next one, or use Show All Entries to display all entries on a single page.


Once finished, use the Breadcrumb Navigation at the top of the page to return to the previous interface.



8. Select the Venue in which you want to Import Materials Request.



9. You will see the list of Logical Spaces already added to this Venue. 


Alternatively, you can get to this interface by selecting Event - Venue in the Navigation Panel under the Material Planning section. 



10. If the Logical Space in which you want to Import Materials Request is already on the list, proceed to step 13. Otherwise, click Add logical space to venue to add a new one.



11. Proceed with this step only if the required Logical Space is missing from the list.


Because the Material Planning Manager can create a Materials Request under any Project, the Project name must be selected. Functional Material Managers assigned to this Project will be able to view and edit this request.


Venue Logistics Managers can only view Material Requests. They do not have permission to add, edit, or delete Material Requests.



12. Once the Project is selected, you will see a list of Logical Spaces that have not yet been added to the selected Venue under this Project. Logical Spaces that have already been added will not appear in this interface.


Use the checkboxes in the Selection Bar to select the Logical Spaces you want to add. You can use the Search and Filters to narrow down your selection. Once finished, click Add logical space to venue.


Note that selections are not saved between pages — if you navigate to another page, your previous selections will be cleared.


To avoid losing selections, finalize your actions on the current page before moving to the next one, or use Show All Entries to display all entries on a single page.


Once finished, use the Breadcrumb Navigation at the top of the page to return to the previous interface.



13. Select the Logical Space in which you want to Import Materials Request.



14. Click Import.



15. Select the source Event, Venue, Logical Space, and Project that submitted the original Material Request and click Import.



16. Status message will be displayed listing any Materials that were not imported.



17. The remaining Materials will be imported under the same Project as the original Material Request.
