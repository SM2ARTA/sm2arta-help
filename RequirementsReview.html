Requirements Review


The Requirements Review is an interactive pivot-style report designed to analyze Material Requirements across multiple Venues and Logical Spaces.


It helps planners quickly identify variations, align quantities, and streamline adjustments — particularly after venue or space duplications.


Purpose and Benefits:


  • Enables quick comparison of Material Requests across similar spaces;
  • Helps normalize quantities after duplication;
  • Reduces manual effort when synchronizing Material Plans across venues;
  • Enhances visibility and accuracy during the review and refinement phase.


In this report:


  • Rows represent Materials;
  • Columns display quantities for the same Logical Space across multiple Venues;
  • Both Venues and Materials can be filtered, allowing targeted comparisons and facilitating bulk updates.


Key Functions:


  • Edit Requested Quantities directly within the pivot grid;
  • Add Materials to the same Logical Space across multiple Venues in one action;
  • Remove Materials simultaneously from multiple spaces or venues;
  • Duplicate Requested Quantities across the same row to quickly align requirements;
  • Access the Material Request Drawer to view or edit details such as Comments, History, Material Information, and Material Notes;
  • Duplicate Logical Space.

Actions & Interface


1. Go to Material Planning - Overview in the Navigation Panel.



2. You will see all non-archive Events created in the system. Functional Material Planners and Venue Logistics Managers will only see the Events they have been granted access to.



3. Click on the Requirements Review icon in the Action Column next to selected Event or click on the Event for which you want to access Requirement Review.



4. Once inside the selected Event, click on the Requirements Plan icon in the Action Bar.


Alternatively, you can get to this interface by selecting Event - Overview in the Navigation Panel under the Material Planning section. 



5. You will see  Requirements Review report for selected Event. 


Use Project Name and Space Name fields to initiate the report.


Category name, Venue and Consumable checkbox can be used to narrow the view.


Analytics (in orange) show total count and sum of all Materials on all pages and will adjust when using Search and Filters.



6. To add Materials, click on Add materials. The process is similar to Add Materials Request. Materials will be added to all visible (filtered) Venues—that is, to all columns currently displayed in the report—if they have not been added before (i.e., if the row is not visible or contains a blank value).


In the Add Materials interface, all Materials - both those already added and those not yet added—are displayed. If a Material already exist in the Venue (appearing in column even with zero value), it will not be added again. Otherwise, the Material will be added automatically to all Venues where it was not previously included (blank columns).



7. To Remove Materials, use the checkboxes in the Selection Bar to select the Material you want to Remove. You can use the Search and Filters to narrow down your selection. Once finished, click Remove materials. Materials will be removed from all visible (filtered) Venues - that is, from all columns currently displayed in the report.


Note that selections are not saved between pages — if you navigate to another page, your previous selections will be cleared.


To avoid losing selections, finalize your actions on the current page before moving to the next one, or use Show All Entries to display all entries on a single page.



8. To edit Requested quantity, click on the value in Venue column. Input new value. Once done, click Accept icon.



9. To duplicate Requested quantity, click on the Duplicate icon and confirm duplication. The Requested quantity will be duplicated hotizontaly across all visible (filtered) columns in the report.



10. To duplicate Logical Space, click the dropdown icon next to Venue name and select Duplicate. Follow the steps of Duplicate Logical Space process.



11. The counters next to the Duplicate icon show the number of comments associated with this Materials Request. Click on the counter to open Materials Request Drawer and reply or edit comments.
