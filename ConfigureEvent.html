Configure Event

Overview


Each Event in SM2ARTA represents a distinct project configuration that may include its own set of Event Components: Venues, Spaces, and Materials.


Events can vary in type and scale — for example, they can be test events, main tournaments, ceremonies, or training events — each requiring different components and user access.


SM2ARTA allows every Event to be configured individually, ensuring full flexibility in managing multiple Events within the same environment.


For each Event, you can:


  • Add and remove Catalogues;
  • Add and remove Venues;
  • Add and remove Logical spaces.


Whatever you add to an Event becomes available for Functional Material Planners for selection and use within the Material Planning module in line with configured User Accesses.


This approach helps Material Planning Managers maintain clear separation between Events while allowing shared resources, such as Venues, Spaces and Materials, to be reused when needed.

Actions & Interface


1. Go to Events - Overview in the Navigation Panel.



2. You will see the full list of Events created in the system. 



3. Click on the Event you want to configure. 



4. Alternatively, you can click on the Event you want to configure directly in the Navigation Panel.



5.You will be redirected to the Event Catalogues interface for the selected Event and see all Catalogues added to the selected Event. 


Use tabs at Tabs Bar to navigate to between other Event Components interfaces.



6. At each tab you can Add or Remove selected Event Components.



7. To Remove a Component, use the checkboxes in the Selection Bar to select the Component you want to Remove. You can use the Search and Filters to narrow down your selection. Once finished, click Remove component.


Note that selections are not saved between pages — if you navigate to another page, your previous selections will be cleared.


To avoid losing selections, finalize your actions on the current page before moving to the next one, or use Show All Entries to display all entries on a single page.



4. To add new Component, click Add component.



5. You will see a list of Components that have not yet been added to selected Event. Components that are already included in a selected Event will not appear in this interface.


Use the checkboxes in the Selection Bar to select the Component you want to add. You can use the Search and Filters to narrow down your selection. Once finished, click Add component.


Note that selections are not saved between pages — if you navigate to another page, your previous selections will be cleared.


To avoid losing selections, finalize your actions on the current page before moving to the next one, or use Show All Entries to display all entries on a single page.


Once finished, use the Breadcrumb Navigation at the top of the page to return to the previous interface.
