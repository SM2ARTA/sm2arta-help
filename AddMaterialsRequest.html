  1. Material Planning

Add Materials Request


1. Go to Material Planning - Overview in the Navigation Panel.



2. You will see all non-archive Events created in the system. Functional Material Plannes and Venue Logistics Managers will only see the Events they have been granted access to.



3. Click on the Event to which you want to Add Materials Request.



4. You will see the list of Venues already added to this Event. 


Alternatively, you can get to this interface by selecting Event - Overview in the Navigation Panel under the Material Planning section. 



5. If the Venue for which you want to Add Materials Request is already on the list, proceed to step 8. Otherwise, click Add venue to add a new one.



6. Proceed with this step only if the required Venue is missing from the list.


Because the Material Planning Manager can create a Materials Request under any Project, the Project name must be selected. Functional Material Managers assigned to this Project will be able to view and edit this request.




7.Once the Project is selected, you will see a list of Venues that have not yet been added to the selected Event under this Project. Venues that have already been added will not appear in this interface.


Use the checkboxes in the Selection Bar to select the Venues you want to add. You can use the Search and Filters to narrow down your selection. Once finished, click Add venue..


Note that selections are not saved between pages — if you navigate to another page, your previous selections will be cleared.


To avoid losing selections, finalize your actions on the current page before moving to the next one, or use Show All Entries to display all records on a single page.


Once finished, use the Breadcrumb Navigation at the top of the page to return to the previous interface.



8. Select the Venue for which you want to Add Materials Request.



9. You will see the list of Logical Spaces already added to this Venue. 


Alternatively, you can get to this interface by selecting Event - Venue in the Navigation Panel under the Material Planning section. 



10. If the Logical Space for which you want to Add Materials Request is already on the list, proceed to step 13. Otherwise, click Add logical space to venue to add a new one.



11. Proceed with this step only if the required Logical Space is missing from the list.


Because the Material Planning Manager can create a Materials Request under any Project, the Project name must be selected. Functional Material Managers assigned to this Project will be able to view and edit this request.



12. Once the Project is selected, you will see a list of Logical Spaces that have not yet been added to the selected Venue under this Project. Logical Spaces that have already been added will not appear in this interface.


Use the checkboxes in the Selection Bar to select the Venues you want to add. You can use the Search and Filters to narrow down your selection. Once finished, click Add logical space to venue..


Note that selections are not saved between pages — if you navigate to another page, your previous selections will be cleared.


To avoid losing selections, finalize your actions on the current page before moving to the next one, or use Show All Entries to display all records on a single page.


Once finished, use the Breadcrumb Navigation at the top of the page to return to the previous interface.



13. Select the Logical Space for which you want to Add Materials Request.



14. Сlick Add materials to space to select Materials.



15. Because the Material Planning Manager can create a Materials Request under any Project, the Project name must be selected. Functional Material Managers assigned to this Project will be able to view and edit this request.



16. Once the Project is selected, you will see a list of Materials that have not yet been added to the selected Logical Space under this Project..Materials that have already been added will not appear in this interface.


Use the checkboxes in the Selection Bar to select the Materials you want to add. Use Qty column to input requested quantities. You can use the Search and Filters to narrow down your selection. 


To view more details about Materials, click the individual Expand icons before the Qty, or use Expand All to open all records. Use the Collapse icon or Collapse All to hide the additional details.


Once finished with selections, click Add materials to space..


Note that selections are not saved between pages — if you navigate to another page, your previous selections will be cleared.


To avoid losing selections, finalize your actions on the current page before moving to the next one, or use Show All Entries to display all records on a single page.


Once finished, use the Breadcrumb Navigation at the top of the page to return to the previous interface.
