Disable User


Users that cannot be deleted but should no longer have access to the system can be disabled.


Disabling a user account immediately prevents login while preserving all historical records, assignments, and audit information associated with that user.


Disabled users can be reactivated at any time if access needs to be restored.


1. Go to Users - Overview in the Navigation Panel.



2. Click the Edit icon in the Action Column next to the User you want to disable.



3. Click Block user.



4. To restore access click Reactivate user.
