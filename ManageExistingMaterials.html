Manage Existing Materials


SM2ARTA allows to record the number of Existing Materials available at each Logical Space.


This helps reflect assets that are already on-site and can reduce the quantities that need to be newly procured or delivered.


Existing Materials are captured as numerical values only. Detailed attributes such as item type, condition, or specifications are not recorded in this field.


It is therefore assumed that any Existing Material can serve as a like-for-like replacement for the corresponding requested item.


Existing Materials directly affect the Required Quantity, which is automatically calculated as:


Required = Requested – Existing


Where:


  • Requested — quantity initially entered by the Functional Material Planner
  • Existing — quantity already available on-site


The system uses this difference to determine how many additional units required to be procured and distributed.


Actions & Interface


1. Go to Material Planning - Overview in the Navigation Panel.



2. You will see all non-archive Events created in the system. Functional Material Planners and Venue Logistics Managers will only see the Events they have been granted access to.



3. Click on the Event in which you want to Manage Existing Materials.



4. You will see the list of Venues added to this Event. 


Alternatively, you can get to this interface by selecting Event - Overview in the Navigation Panel under the Material Planning section. 



5. Select the Venue in which you want to Manage Existing Materials.



6. You will see the list of Logical Spaces added to this Venue. 


Alternatively, you can get to this interface by selecting Event - Venue in the Navigation Panel under the Material Planning section. 



7. Select the Logical Space in which you want to Manage Existing Materials.



8. You will see the list of Materials added to this Logical Space. 



9. Click on the Existing quantity to Add, Edit or Remove it. Once done, click the Accept icon.
