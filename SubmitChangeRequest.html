Submit Change Request


Change Requests can be submitted for both existing Material Request lines and new Materials within existing Logical Spaces. This allows Functional Material Planners to request adjustments to quantities or add additional materials as operational needs evolve.


However, submitting requests for new Logical Spaces is not part of the automated Change Management workflow. If a new Logical Space needs to be added after planning has been blocked, it must be done through a manual request submitted to the Material Planning Manager for review and approval.


Material Planning Managers cannot submit new Material Requests within existing Logical Spaces while Change Management is active. Any new materials they add will be treated as regular Material Requests, not as part of the Change Management process.

To perform this action within the Change Management workflow, the Material Planning Manager should use the Login As Functional Material Planner function and submit the request under the corresponding Project.

Actions & Interface


1. Go to Material Planning - Overview in the Navigation Panel. You will see all non-archive Events created in the system. 


Functional Material Planners and Venue Logistics Managers will only see the Events they have been granted access to.



2. Click on the Event for which you want to Submit Change Request.


The Event must be Blocked (grey-scaled).



3. You will see the list of Venues added to this Event. 


Alternatively, you can get to this interface by selecting Event - Overview in the Navigation Panel under the Material Planning section. 



4. Select the Venue in which you want to Submit Change Request.



5. You will see the list of Logical Spaces added to this Venue. 


Alternatively, you can get to this interface by selecting Event - Venue in the Navigation Panel under the Material Planning section. 



6. Select the Logical Space in which you want to Submit Change Request.



7. You will see the list of Materials added to this Logical Space. 


Requested column is the initial request from Functional Area, Existing column is used to capture existing Materials in the Space and Required is the difference between Requested and Existing.


Change request column is used to submit Change Request.



8. Click on the Change request quantity to Add or Edit it. Once done, click the Accept icon.


Venue Logistics Managers can only view Change Requests. They do not have permission to add, edit, or delete Change Requests.



9. To submit a new Materials Request, first Login As Functional Material Planner, then navigate to the Logical Space where you want to submit the new request. Click Add change request to space. The process is similar to Add Materials Request, but all new entries will be recorded under the Change request column.


Until the new Materials Request is approved, Functional Material Planner can remove it.
