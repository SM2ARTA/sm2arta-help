Change Management Report


The Change Management Report provides a consolidated view of all active Change Requests within the event, grouped by Cluster and Venues.


It serves as the main tool for Material Planning Managers to analyze, balance, and approve changes efficiently across multiple sites.


Key Columns:


  • Available materials - Shows the accumulated number of materials released through Decrease Change Requests that can be used to offset other requests;
  • Delta - Displays the consolidated difference across all Change Requests for this Material within the Venue or Cluster:
    • Positive Delta - decrease in Requested materials;
    • Negative Delta - increase in Requested materials.


The Action Column provides the option for Bulk Approval when there are sufficient Available materials to cover pending Change Requests. When triggered, the system automatically approves each individual Change Request sequentially.


  • If Available materials are insufficient, bulk approval will not be active;
  • Once Change Requests are approved, related Material Request lines are updated in the Material Plan, and the report recalculates Delta and Available Materials in real time;
  • When all Change Requests for a Material are processed, the line is automatically removed from the Change Management Report to maintain a clear and current overview.
  • Decline is not available at this interface and needs to be handled through Material Plan report.

Actions & Interface


1. Go to Material Planning - Overview in the Navigation Panel.



2. You will see all non-archive Events created in the system. Functional Material Planners and Venue Logistics Managers will only see the Events they have been granted access to.



3. Click on the Change Management Report icon in the Action Column next to selected Event or click on the Event for which you want to access Change Management Report.


The Change Management Report will be available only for Blocked events, where Change Management is Enabled.



4. Once inside the selected Event, click on the Change Management Report icon in the Action Bar.


Alternatively, you can get to this interface by selecting Event - Overview in the Navigation Panel under the Material Planning section. 



5. You will see  Change Management Report for selected Event. 


Available materials - Shows the accumulated number of materials released through Decrease Change Requests that can be used to offset other requests.


Delta - Displays the consolidated difference across all Change Requests for this Material within the Venue or Cluster:

  • Positive Delta - decrease in Requested materials;
  • Negative Delta - increase in Requested materials.



6. Use Confirm icon in the Action Column to initiate bulk approval of Change Requests. Status message will be displayed.
