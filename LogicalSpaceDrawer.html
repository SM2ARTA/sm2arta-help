Logical Space Drawer


The Logical Space Drawer is the main interface for managing space-level planning details within SM2ARTA.


It provides Material Planning Managers and authorized users with direct access to all information and tools associated with a specific Logical Space — enabling quick review, updates, and documentation without leaving the main Material Planning view.


Accessible Features:

  • Link physical space;
  • Manage bump-in & bump-out dates;
  • Flag Overlay space;
  • Manage Logical Space note;
  • View and upload images.

Actions & Interface


1. Go to Material Planning - Overview in the Navigation Panel.



2. You will see all non-archive Events created in the system. Functional Material Planners and Venue Logistics Managers will only see the Events they have been granted access to.



3. Click on the Event in which you want to access Logical Space Drawer.



4. You will see the list of Venues added to this Event. 


Alternatively, you can get to this interface by selecting Event - Overview in the Navigation Panel under the Material Planning section. 



5. Select the Venue in which you want to access Logical Space Drawer.



6. You will see the list of Logical Spaces added to this Venue. 


Alternatively, you can get to this interface by selecting Event - Venue in the Navigation Panel under the Material Planning section. 



7. Click on the Info icon next the selected Logical Space to open the Logical Space Drawer.



8. Logical Space Drawer will open on the right.



9. To add Physical Space, click Add physical space and select the physical space name from drop-down menu. The physical space should be precreated in the system before it can be linked. Once done, click Save.


To remove Physical space, click on Delete icon.



10. To manage Bump-in and Bump-out dates, click on the (not set) or date field. Once the date is selected, click on Accept icon. Duration will be calculated automaticaly.


To remove the date, empty the field and click Accept icon.



11. To flag/unflag Overlay space, check or uncheck the Overlay checkbox. The status message will be displayed accordingly.



12. To manage Logical Space Note, select tab Note in the Tabs Bar. Add Note and click Add. The status message will be displayed accordingly.


To edit Note, type the new Note and click Update.The status message will be displayed accordingly.


To remove Note, clear the field and click Update. The status message will be displayed accordingly.



13. To manage Logical Space Images, select tab Images in the Tabs Bar. Attach the image and click Upload.The status message will be displayed accordingly. Repeat as required.


To remove Image, click on the Delete icon.
