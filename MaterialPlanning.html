Material Planning


Overview


The Material Planning module is the foundation for building event-specific material plans in SM2ARTA. It allows Users to define and organize all material requirements for each Event, ensuring traceability, collaboration, and alignment across multiple teams.


Material Planning is structured around three core steps:


  • Add Venue – Select the venue where materials will be used
    • Add Logical Spaces – Define the spaces within each venue (e.g., Media Center, Volunteer Lounge, Team Dressing Room)
      • Add Materials from Catalogue – Populate each logical space with materials from the materials catalogue


Collaborative Structure


All material requests are made under Projects to which users are assigned. This ensures that:


  • Users within the same project can view and build upon each other’s work;
  • Collaboration between Functional Material Planners happens seamlessly within one consolidated master file;
  • Changes and additions made by any user are automatically synchronized across the shared event structure.


Material Planning Managers have unrestricted access and can create or edit material requests under any project.


Usability Features


SM2ARTA provides a range of built-in tools to analyze, refine, and optimize planning, including:


  • Historical Data Import – Retrieve and import planning data from previous events for reuse at current event;
  • Duplication – copy the planning between spaces or venues;
  • Requirements Review – Evaluate material requirements across multiple venues for selected Projects and Space;
  • Planning Validation – Dashboard to verify data consistency, completeness, and budget alignment;
  • Comments - Add internal communication notes between planners directly within the Material Plan for easier collaboration and reviews;
  • Notes - Record important references, reminders, or specific instructions related to Materials or Spaces for future review.


Advanced Planning Features


As planning progresses, users can enrich the material plan with additional details, such as:


  • Existing Materials - Identify and account items already available at the venue to optimize reuse;
  • Bump-In and Bump-Out Dates – Define delivery and removal timelines for each item;
  • Link to Physical Spaces – Associate logical spaces with corresponding physical space data;
  • Floorplans and Photos – Upload reference layouts and images for better visualization.
