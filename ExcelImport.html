Excel Import


SM2ARTA allows users to bulk import Material Requests directly from an Excel file, enabling a fast and efficient transition from spreadsheet-based planning into the SM2ARTA system.


This feature is particularly useful for events where initial planning was started in Excel, allowing existing data to be seamlessly integrated into the centralized platform.


SM2ARTA requires the import file to follow a specific template format.


To maintain data accuracy and integrity, the following condition must be met before import:


  • All Event Components referenced in the import file (e.g., Projects, Venues, Logical Spaces, and Materials) must already exist in the SM2ARTA Directory;
  • If any referenced component does not exist in the Directory, it will not be imported or automatically created by the system;
  • If any Material Requests already exist for the same Event, they will be overwritten during the import process.


The imported file will therefore represent the new, consolidated Material Plan for that Event and ensures that all imported data is properly mapped to the corresponding fields within the system, including Projects, Venues, Logical Spaces, and Materials.


When the import is completed successfully, SM2ARTA automatically configures the Event as follows:

  • Materials are added to the system under a dedicated catalogue named “Imported Goods” and linked to destination Event;
  • Venues and Logical Spaces listed in the Excel file are linked to the destination Event.


This automation ensures that the imported data integrates smoothly into the event environment without additional setup steps.


Import is performed through Material Plan report covered in details in this section.


Actions & Interface


1. Go to Material Planning - Overview in the Navigation Panel.



2. You will see all non-archive Events created in the system. Functional Material Planners and Venue Logistics Managers will only see the Events they have been granted access to.



3. Click on the Material Plan icon next to selected Event or click on the Event for which you want to access Material Plan.



4. Once inside the selected Event, click on the Material Plan icon.


Alternatively, you can get to this interface by selecting Event - Overview in the Navigation Panel under the Material Planning section. 



5. You will see consolidated Material Plan report for selected Event.



6. To import Excel, click the Import icon.



7. Select the file and click Upload icon.



8. Once the upload is complete, the progress bar will show 100%. Click Press to complete to finish Import.



9. You will see the status message.
