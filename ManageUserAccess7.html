Manage User Access


1. Go to Spaces - Logical spaces - Overview in the Navigation Panel.



2. Click the Access icon in the Action Column next to the Logical Space you want to manage access for.



3. You will be redirected to the User Access interface for the selected Logical Space, where you can view all Users that currently have access to it. Note that Material Planning Managers are not listed as they have access to all system features by default. Though Venue Logistics Managers are listed, they have access to all Logical Spaces attached to the Events they have access to.



4. You can Add access or Remove access for selected Logical Space by adding or removing Users to/from it.



5. To Remove User Access, use the checkboxes in the Selection Bar to select the Users you want to remove access from. You can use the Search and Filters to narrow down your selection. Once finished, click Remove access.


Note that selections are not saved between pages — if you navigate to another page, your previous selections will be cleared.


To avoid losing selections, finalize your actions on the current page before moving to the next one, or use Show All Entries to display all records on a single page.



6. To add new User Access, click Add access.



7. You will see a list of Users that does not yet have access to selected Logical Space. Users that already has access will not appear in this interface.


Use the checkboxes in the Selection Bar to select the Users you want to add access to. You can use the Search and Filters to narrow down your selection. Once finished, click Add access.


Note that selections are not saved between pages — if you navigate to another page, your previous selections will be cleared.


To avoid losing selections, finalize your actions on the current page before moving to the next one, or use Show All Entries to display all records on a single page.


Once finished, use the Breadcrumb Navigation at the top of the page to return to the previous interface.
